📝 Descripción
En este reto deberás desarrollar una página de contacto utilizando HTML y CSS. El objetivo principal es practicar el uso de formularios, campos de entrada e inputs, así como aplicar un diseño responsivo y visualmente atractivo.




🎯 Objetivos
Con esta actividad practicarás:

El diseño de formularios responsivos con HTML y CSS.
El uso de inputs, placeholders y menús desplegables (dropdowns).
La creación de una página de contacto completa.
La aplicación de estilos CSS para lograr un diseño profesional.




🧩 Requerimientos
Tu página debe incluir los siguientes elementos:

Un ícono representativo.
Un encabezado (heading) y una breve descripción.
Un formulario con:
Campos de entrada con placeholders.
Un menú desplegable (dropdown) con al menos 2 opciones (la opción por defecto debe ser la indicada en el diseño).
Un botón para enviar el formulario.
Estilos CSS que cuiden espaciado y tipografía, aplicando el siguiente box-shadow (puedes usarlo o personalizarlo):
box-shadow: 0px 30px 100px 0px rgba(17, 23, 41, 0.15);



        
⚙️ Detalles técnicos
Usa únicamente HTML y CSS.
Trabaja con cualquier editor de texto o IDE de tu preferencia.
